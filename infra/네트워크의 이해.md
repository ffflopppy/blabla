
# OSI 7계층 (L4, L7이 뭔지)
~~~
7. 애플리케이션 (HTTP, FTP)      ← L7
6. 표현
5. 세션
4. 전송 (TCP, UDP)             ← L4
3. 네트워크 (IP)                ← L3
2. 데이터링크 (이더넷)
1. 물리 (케이블, 전기신호)
~~~


# TCP/UDP 차이
### TCP
- HTTP, SSH, DB 연결에 사용

### UDP
- 그냥 보내고 끝, 확인 안함.
- 빠르지만 유실 가능.
- 동영상 스트리밍, 게임에 사용.

# IP, 포트 개념
### IP 서버의 주소
192.168.1.1

### 서버 한 대에서 여러 프로그램이 동시에 실행가능
- 192.168.1.1:8080 → Java 앱
- 192.168.1.1:3306 → MySQL
- 192.168.1.1:22   → SSH

#사설 IP vs 공인 IP
### 사설 IP
- 사설 IP == 내부망
- 외부망에서는 직접 접근 불가.
- 약속된 대역이 있음(RFC 1918에 의해 예약된 주소 대역)

### 공인 IP
- 전 세계적으로 유일한 주소
- ISP (KT, SKT, LG)가 할당
